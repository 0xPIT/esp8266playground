# NTP Analog Clock

Simple analog clock face, synchronized via NTP


## Wiring

I've used a NodeMCU module. Wiring:

Display | NodeMCU
--------|---------
LED     | +3V3
SCK     | D5
SDA     | D7
A0      | D4
RESET   | +3V3
CS      | D8
GND     | GND
VCC     | +3V3
